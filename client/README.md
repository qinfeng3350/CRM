# CRM系统前端

## 技术栈

- React 19
- Vite
- Ant Design 5
- React Router 7
- Axios
- Day.js

## 功能模块

1. **认证模块** - 登录、注册
2. **工作台** - 数据概览
3. **客户管理** - 客户CRUD、认领、退回
4. **商机管理** - 商机管理、状态跟踪
5. **合同管理** - 合同管理、审批流程
6. **市场管理** - 线索管理、市场活动
7. **服务管理** - 工单管理
8. **销售管理** - 销售团队、业绩统计
9. **数据分析** - 各类报表分析
10. **系统管理** - 角色、用户管理

## 启动项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 访问地址

开发环境: http://localhost:5173

## 注意事项

1. 确保后端服务运行在 http://localhost:3000
2. 首次使用需要先注册账号
3. 不同角色有不同的权限和功能访问范围
